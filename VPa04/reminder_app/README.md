# Desktop-напоминалка (Windows 11)

Простое desktop-приложение на Python и Tkinter для работы с напоминаниями.

## Структура проекта

- `database.py` - класс `ReminderDatabase` для SQLite3:
  - инициализация базы и таблицы;
  - добавление, удаление, чтение записей;
  - смена статуса (`Ожидает`, `Готово`, `Просрочено`, `Отменено`);
  - автообновление просроченных записей;
  - подсчет статистики.
- `notifications.py` - класс `NotificationManager`:
  - проверка базы каждую секунду;
  - показ системных Windows Toast уведомлений;
  - fallback на popup поверх всех окон;
  - ручной тест уведомлений.
- `gui.py` - графический интерфейс:
  - форма добавления напоминания;
  - быстрые кнопки времени (`1`, `5`, `15` минут, `1` час);
  - таблица с фильтром по статусу;
  - кнопки: добавить, тест уведомления, отметить готово, отменить, удалить;
  - просмотр деталей по двойному клику;
  - подтверждение выхода.
- `main.py` - точка входа.

## Установка

```bash
pip install win10toast
```

> `tkinter` и `sqlite3` обычно уже встроены в стандартный Python для Windows.

## Запуск

```bash
python main.py
```

## Как это работает

1. При старте создается файл `reminders.db` (если его нет).
2. Новые напоминания сохраняются в SQLite.
3. Фоновый поток раз в секунду проверяет, какие напоминания пора показать.
4. Уведомление показывается через Toast (или popup, если Toast недоступен).
5. После показа статус напоминания меняется на `Готово`.

## Формат даты

Поле даты/времени в интерфейсе:

`YYYY-MM-DD HH:MM`  
Пример: `2026-04-10 19:45`
